## 빠른 선택 가이드

### "최고의 토론 품질을 원한다"
→ **Cohere Command A Reasoning**
- 256K 컨텍스트로 전체 대화 유지
- 고급 추론으로 논리적 허점 지적
- $2.50/$10 가격

### "비용을 최소화하고 싶다"
→ **AI21 Jamba Large**
- $2.00/$8.00으로 가장 저렴
- Cohere와 동일한 256K 컨텍스트
- 빠른 응답 속도

### "실시간 사실 검증이 필요하다"
→ **Perplexity Sonar Pro** (보조용)
- 웹 검색으로 증거 수집
- ACT II 비판 단계에만 사용
- 월 $20 기본 + 크레딧

### "모든 기능을 다 하고 싶다"
→ **하지 마라** (Reka는 오버스펙)
- 불필요한 멀티모달 기능
- 토론에는 과도한 비용 ($27/월)
- 컨텍스트가 오히려 부족 (4K-8K)
